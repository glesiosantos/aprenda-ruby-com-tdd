# Referencias

Este livro usa referencias publicas para orientar ordem, ferramentas e nomenclatura. A proposta e original para Ruby, mas reconhece explicitamente as fontes que inspiraram a estrutura.

## Inspiracao didatica

- Aprenda Go com Testes: https://larien.gitbook.io/aprenda-go-com-testes
- Learn Go with Tests, versao original em ingles: https://quii.gitbook.io/learn-go-with-tests

A inspiracao principal e o formato: aprender uma linguagem em passos pequenos, sempre escrevendo testes antes da implementacao. Os exemplos, textos e escolhas de Ruby deste repositorio sao proprios.
